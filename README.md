# Termux TSU Patch

Quick patch for `tsu` in Termux to fix or update the executable.

## Requirements

- Termux
- `tsu`, `patch` (will be installed automatically if missing)

## Usage

Run this command in Termux:

```bash
curl -sL https://raw.githubusercontent.com/Alexjr2/magisk_alpha_fix_termux_tsu/main/tsu.sh | bash
